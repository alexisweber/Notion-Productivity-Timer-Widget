# Compact-Notion-Productivity-Timer-Widget
### Copy the link to the widget, paste in your Notion page and select Embed.

Compact productivity timer widget for Burst focus (preset 25minutes, Pomodoro) or Deep focus (preset 2 hours). Utilizes circle countdown and various sounds. Customizable color themes (pink, blue, yellow, black/white) and timer sounds (soft, loud, birds, music).
